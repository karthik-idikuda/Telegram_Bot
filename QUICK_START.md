## 🚨 **TROUBLESHOOTING**

**Bot not responding?**
- Check if bot is running (should see "Application started" in terminal)
- Restart: Stop terminal, run `python bot.py` again

**Voice not working?**
- Need OpenAI API key in .env (optional feature)
- Text commands work perfectly!

**Want to restart fresh?**
- Delete JSON files in project folder
- Restart bot
