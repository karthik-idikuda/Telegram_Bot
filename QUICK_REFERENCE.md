# 🚀 SycproBot - Quick Reference Card

## 📝 **Commands**

```
/start          - Welcome & overview
/addtask        - Add smart task
/mytasks        - View all tasks
/stats          - Progress stats
/goal           - Break down goal
/moodreport     - Weekly report
/help           - Get help
```

## 💬 **Smart Task Examples**

```
/addtask Urgent: Submit report today
→ 🔴 HIGH priority, due today

/addtask Call mom every Sunday at 6 PM
→ 🔄 Recurring weekly

/addtask Workout tomorrow at 7 AM
→ ⏰ Scheduled reminder

/addtask Maybe learn guitar someday
→ 🟢 LOW priority
```
